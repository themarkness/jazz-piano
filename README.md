# Jazz Keys Lab

Jazz Keys Lab is a mobile-friendly practice studio that helps you learn jazz
chords on piano. Explore chord fingerings, hear lush voicings, practice common
progressions, and check your note choices with simple chord recognition
feedback.

## Features

- **Chord fingerings** for left and right hands.
- **Chord playback** with built-in audio previews.
- **Progression trainer** for II-V-I and other jazz staples.
- **Chord recognition** to verify the notes you play.
- **Pop reharmonization** ideas to connect jazz harmony to familiar songs.

## Running locally

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the development server:
   ```bash
   npm run dev
   ```
3. Open `http://localhost:3000` in your browser.
